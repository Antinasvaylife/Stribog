# Утилита для хэширования по ГОСТ-34.11 "Стрибог"
Кроссплатформенное приложение для вычисления хэша файла или строки по ГОСТ-34.11.

## Установка

1. Клонирование репозитория 

```git clone https://github.com/Antinasvaylife/Stribog.git```

2. Сборка из исходников

```mkdir build_cmake && cd build_cmake```
```cmake ..```
```cmake --build . --config <Configuration>```
```cd <Configuration>```

## Использование

1. Вывести в консоль справочное сообщение

```Stribog -h```

2. Вывести в консоль 512-битный хэш строки или файла

```Stribog -s <string_to_hash>```
```Stribog -f <path_to_file>```
